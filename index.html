2. Firebase (חינם עד 1GB):
→ console.firebase.google.com → New Project → "wedding-app"
→ Realtime Database → Create → Test mode
→ Project Settings (⚙️) → Your apps → </> → Copy config
→ הכנס apiKey, projectId, databaseURL

3. Google Drive API:
→ console.cloud.google.com → New Project
→ APIs & Services → Enable → "Google Drive API"
→ Credentials → Create → OAuth 2.0 Client ID → Web App
→ Authorized origins: כתובת האתר שלך ב-GitHub Pages
→ Copy Client ID
